Задание 04: Сканирование и предобработка текста (OCR)

Выполнил Селиванов Иван, группа ПИ51

Описание: Подготовьте лист бумаги с текстом и снимите его под углом. Напишите программу для автоматического определения четырех углов листа. Используйте перспективное преобразование (Perspective Transform) для выравнивания изображения и бинаризуйте его для четкости текста.

Рекомендуемая ОС: Windows / Linux / macOS (у меня сейчас)

Окружение: Python 3.x (pip install opencv-python numpy)
